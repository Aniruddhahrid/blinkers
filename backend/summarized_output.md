
				CARD ><!-- Start of the content --}## 7.2 System vs. Model: Evolving Roles in Agentic AI {#system-vs-model-evolving-roles-in-agentic-ai .section-header}### Overview of Current Engineering Focus {#overview-of-current-engineering-focus .subsection-header}[51] highlights a common sentiment among industry practitioners that production-ready agentic systems are predominantly '90% engineering, 10% intelligence.' This pragmatic stance underscores the current reality where AI is often seen as a functional component or specialized tool rather than the central driver of the system itself. However, this perspective is expected to evolve as underlying technologies mature and become more integrated into the core functionalities of agentic systems. The historical analogy with the evolution of web applications provides valuable insights: just as the internet transitioned from purely engineering-driven stages to a balance of engineering and intelligence, we anticipate similar advancements in agentic AI. This section outlines three key stages in this anticipated evolution: pipeline design (2023-2025), model-native transition (2025-2027), and autonomous evolution (post-2027). Each stage represents a shift in the balance between engineering and intelligence, leading to increasingly self-sufficient agentic models. {#system-vs-model-evolving-roles-in-agentic-ai .section-header}#### Pipeline Design (2023-2025) {#pipeline-design-2023-2025 .subsection-header}In the current era, large language models (LLMs) are often treated as components within a human-designed pipeline. This stage emphasizes the integration of LLMs into existing workflows and processes, leveraging their capabilities to enhance specific tasks or functionalities. As these systems become more prevalent, they will likely be integrated into broader application frameworks, such as LangChain, AutoGen, AgentCore, and Agent Builder, which serve as intermediate layers bridging model-native intelligence with real-world deployment. {#pipeline-design-2023-2025 .subsection-header}#### Model-Native Transition (2025-2027) {#model-native-transition-2025-2027 .subsection-header}This phase marks a significant shift towards model-driven core agentic functions, including planning, tool use, and memory. As models increasingly internalize these capabilities, the role of the system layer evolves from compensating for deficits to providing foundational ecosystem support. For instance, in the realm of dialogue, traditional systems often required separate modules for intent understanding, dialogue management, and response generation. With advancements like ChatGPT, model layers have integrated this entire flow internally. Consequently, the system layer's role shifts towards supporting external data access and context management through frameworks such as LangChain and LlamaIndex. Similarly, in planning, models are increasingly internalizing reasoning processes through reinforcement learning (RL), necessitating robust evaluation frameworks provided by platforms like OpenAI and Statsig. For tool use, models that can plan and execute tool calls require a standardized ecosystem for integration, exemplified by MCP-based tool transformation and scalable server integrations (e.g., Fixie.ai, Compositio). {#model-native-transition-2025-2027 .subsection-header}#### Autonomous Evolution (Post-2027) {#autonomous-evolution-post-2027 .subsection-header}In this final stage, the focus shifts to fostering highly autonomous agents capable of emergent task discovery, on-demand AI-generated software, and even architectural self-evolution. As agentic models become more self-sufficient, the role of the system layer will transition from compensating for deficits to providing foundational ecosystem support. For example, in memory management, the current reliance on external vector databases and context engineering (e.g., Letta, Pinecone) will evolve into supporting model-native memory through cross-platform synchronization services, data compliance, and privacy controls. In multi-agent collaboration, pre-defined roles and externally orchestrated workflows (e.g., CrewAI, LangGraph) will give way to emergent collaboration platforms that manage environment and resource scheduling, inter-agent communication, and collaborative evaluation. {#autonomous-evolution-post-2027 .subsection-header}### The Role of the System Layer in Future Agentic AI {#the-role-of-the-system-layer-in-future-agentic-ai .subsection-header}As core agentic capabilities are progressively internalized, the system layer will evolve from its current state as a collection of capability-specific patches into a foundational AgentOps infrastructure. This future system layer will focus on providing domain-agnostic, scalable services essential for a robust agent ecosystem, evolving existing infrastructure to meet new, autonomous demands. Key areas include: {#the-role-of-the-system-layer-in-future-agentic-ai .subsection-header}#### Agent Identity Management {#agent-identity-management .subsubsection-header}The paradigm for human/device identity and authorization (e.g., Okta) will be extended to manage agent identity, enabling secure, auditable authentication for autonomous entities. {#agent-identity-management .subsubsection-header}#### Agent Communication {#agent-communication .subsubsection-header}Infrastructure for human-to-human communication (e.g., Twilio) will evolve to support high-bandwidth, structured agent-to-agent communication protocols. {#agent-communication .subsubsection-header}#### Agent Execution Observability {#agent-execution-observability .subsubsection-header}The focus will expand from traditional system-level observability (e.g., Datadog) to the much more complex challenge of agent execution process observability, which involves tracing and debugging the emergent, non-deterministic decision-making paths of autonomous agents. {#agent-execution-observability .subsubsection-header}### Conclusion {#conclusion .section-header}The evolution of agentic AI reflects a profound transformation in how intelligence is conceived, trained, and deployed. From pipeline-based systems that orchestrate reasoning, memory, and action through external scaffolds to model-native paradigms that internalize these capabilities, we are witnessing a fundamental redefinition of agentic AI. Reinforcement learning acts as the engine of experience, bridging perception and action, turning static models into adaptive, goal-directed entities capable of learning from interactions with the environment.
